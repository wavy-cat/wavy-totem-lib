### Написание простой программы

Для того чтобы убедиться что библиотека установлена и работает корректно, а также для небольшой демонстрации работы, напишите простую программу:

```python
from wavy_totem_lib import TotemBuilder, Skin

# Создайте экземпляр класса Skin с файлом вашего скина
skin = Skin('my_skin.png')

# Создайте билдер тотема на основе экземпляра Skin
builder = TotemBuilder(
    skin,
    round_head=True  # Опционально: закруглите края головы
)

# Соберите тотем
totem = builder.build()

# Сохраните изображение тотема в файле my_totem.png
totem.image.save('my_totem.png')
```

Данный код загрузит [ваш скин](/ru/guides/downloading-the-skin) из файла `my_skin.png`, сгенерирует тотем и сохранит его в файле `my_totem.png`.

Если вы смогли увидеть ваш тотем, значит библиотека исправно работает, продолжайте изучать документацию дальше!
