Skin — это полезный для паттернов тип, который представляет сырое изображение скина в виде отдельных частей тела с доступом через свойства, а также даёт метаданные скина.

## Инициализация

```py
from wavy_totem_lib import Skin

skin = Skin(
    filepath='my_skin.png',
    slim=True
)
```

Skin принимает 2 аргумента:

* `filepath`: `Union[str, bytes, Path, IO[bytes]]` — изображение скина в виде bytes или путь к нему. Передаётся в неизменном виде в `PIL.Image.open()`.
* `slim`: `bool` (Опциональный. По-умолчанию является EllipsisType) — булево значение, указывающее является ли скин узким. Если передан EllipsisType (aka `...`), тип попытается самостоятельно определить размер скина.

### Свойства с метаданными

* `image`: `PIL.Image` — содержит всё изображение целиком. Принудительно использует цветовой профиль RGBA.
* `version`: `str` — версия скина. Может иметь значение `new` для скинов 64x64, которые поддерживают верхний слой, и `old` для скинов 64x32, которые **не** поддерживают его.
* `available_second`: `bool` — указывает на то, поддерживает ли скин верхний слой. Значение зависит от `version`.
* `is_slim`: `bool` — указывает на то, является ли скин узким. Это значение можно задать принудительно при инициализации.

### Свойства, содержащие части тела

Все эти "свойства" на самом деле являются методами с декоратором `@property` для экономии памяти и процессорного времени.

* `right_leg` — правая нога.

  Отдаёт словарь с ключами `front`, `back`, `left`, `right`, `top`, `bottom` и PIL.Image в качестве значения.

* `right_leg_front` — перед правой ноги.
  
  Отдаёт PIL.Image. Равнозначно `right_leg()['front']`, однако использует меньше ресурсов, т.к. не считает все стороны.

* `right_leg_second` — верхний слой правой ноги.

  Отдаёт словарь с ключами `front`, `back`, `left`, `right`, `top`, `bottom` и PIL.Image в качестве значения, если верхний слой доступен. В противном случае возвращает `None`.

* `right_leg_second_front` — перед верхнего слоя правой ноги.

  Отдаёт PIL.Image, если верхний слой доступен, в противном случае `None`. Равнозначно `right_leg_second()['front']` если верхний слой доступен, однако использует меньше ресурсов.

* `left_leg` — левая нога.

  Отдаёт словарь с ключами `front`, `back`, `left`, `right`, `top`, `bottom` и PIL.Image в качестве значения. Для старых скинов возвращает то же, что и `right_leg`.

* `left_leg_front` — перед левой ноги.

  Отдаёт PIL.Image. Для старых скинов возвращает то же, что и `right_leg_front`. Равнозначно `left_leg()['front']`, однако использует меньше ресурсов.

* `left_leg_second` — верхний слой левой ноги.

  Отдаёт словарь с ключами `front`, `back`, `left`, `right`, `top`, `bottom` и PIL.Image в качестве значения, если верхний слой доступен. В противном случае возвращает `None`.

* `left_leg_second_front` — перед верхнего слоя левой ноги.

  Отдаёт PIL.Image, если верхний слой доступен, в противном случае `None`. Равнозначно `left_leg_second()['front']` если верхний слой доступен, однако использует меньше ресурсов.

* `right_hand_second_front` — перед верхнего слоя правой руки.

  Отдаёт PIL.Image, если верхний слой доступен, в противном случае `None`. Учитывает тип скина (slim/wide). Равнозначно `right_hand_second()['front']` если верхний слой доступен, однако использует меньше ресурсов.

* `body` — тело.

  Отдаёт словарь с ключами `front`, `back`, `left`, `right`, `top`, `bottom` и PIL.Image в качестве значения.

* `body_front` — перед тела.

  Отдаёт PIL.Image. Равнозначно `body()['front']`, однако использует меньше ресурсов.
